Devine's Ethereal Lyricist

Project Site Plan | Devine Phoebe

Site Name

Name: Devine's Ethereal Lyricist

Description: I chose this name to reflect a "heavenly" (Ethereal) approach to songwriting (Lyricist). It provides a strong, artistic brand identity that fits my passion for inspirational poetry and music.

Site Purpose

The purpose of this site is to showcase a personal collection of original poetry and song lyrics. It serves as a creative portfolio that uses dynamic web tools to help visitors filter content by theme and view estimated reading times for each piece.

Scenarios

  • "I am looking for an Inspirational poem to start my morning; can I filter the list to find one quickly?"
  • "I have a short break; which lyrics can I read in under 90 seconds?"

Color Schema

Berry Red
Cream
Sand Tan

The Berry Red will be used for headers and buttons. The Cream is for the overall background, and the Sand Tan will be used for accent borders and decorative cards.

Typography

Headings: Playfair Display (Serif) - Creates an elegant, literary feel.

Body: Roboto (Sans-serif) - Ensures clear, professional readability for song lyrics.
